## K Means Cluster Astrosat

**K-means-Cluster** is a Python-based utility to convert Astrosat fits images into power-density, Hardness intenisty, Fourier lag, Light curve graphs **


---
## How to Use


```bash
python -m venv venv
source venv/bin/activate
python final.py
```
